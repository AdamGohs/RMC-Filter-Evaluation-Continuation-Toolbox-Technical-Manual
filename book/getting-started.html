RMC Filter Evaluation (Continuation) Toolbox Technical Manual

3.1 Getting Started

Copy or download the toolbox file to the computer. To open the toolbox file, either:

  • Find the file on the computer and double-click it. This opens the file in Microsoft Excel.

  • Open Microsoft Excel and use the application to open the file: Once Microsoft Excel is open, go to the File menu at the top of the window and select Open.

The toolbox is an Excel binary workbook (.xlsb) that uses macros. You may need to enable the macros, either before opening the file or by clicking “Enable Content” in the yellow Security Warning message bar with a shield icon that appears after the file is opened. The actual message in the message bar will vary depending on the computer’s settings and installed add-ins. Figure 3.1 displays examples of different wordings that may appear in the message bar.
